# Fitur Utama
- Autentikasi pengguna menggunakan JWT (JSON Web Token)
- Registrasi, Login, Logout, serta pengambilan data profil pengguna
- Proteksi halaman dan REST API menggunakan Protected Routes dan Authentication Middleware
- Manajemen produk (Tambah, Ubah, Hapus (Soft Delete), dan Lihat Produk)
- Pencarian produk berdasarkan nama maupun tanggal
- Sistem kasir dengan keranjang belanja (Shopping Cart)
- Pengaturan jumlah pembelian secara dinamis dengan validasi stok
- Perhitungan total transaksi secara otomatis di sisi server
- Validasi stok untuk mencegah pembelian melebihi stok tersedia
- Riwayat transaksi lengkap beserta detail setiap item transaksi
- Dashboard laporan bisnis dengan ringkasan performa penjualan
- Analisis produk terlaris dan produk dengan penjualan terendah
- Analisis produk dengan harga tertinggi dan terendah
- Monitoring stok produk serta deteksi stok rendah
- Fitur restock produk dengan pencatatan histori stok otomatis
- Stock Log untuk mencatat seluruh aktivitas stok masuk dan keluar
- Pencarian transaksi dan riwayat restock berdasarkan tanggal
- Auto Redirect ketika token kedaluwarsa melalui Axios Interceptor
- Tampilan responsif untuk desktop maupun perangkat mobile
- Animasi antarmuka menggunakan Framer Motion
- REST API dengan arsitektur MVC
